The Hospital Management System it is simple web application develop in PHP MySQL database, using Bootstrap, Modal, HTML, CSS, and Ajax, the system contains of admin and user, the admin can manage all user like update information, delete, and add user. The user is represent a Doctor and Patient. This system web application is very user-friendly and easy to use.

** Admin Dashboard **
Profile, Patients Book, Add Doctors, Doctor’s Record, Appointment, Add Outbreaks, Like Outbreaks, HIV, HIV Reports.

** Profile**
First Name Second Name, Email, Role, Gender, and Phone Number.

** Patients **
Name, Location, Age, Attended, Doctor, and Print.

** Add Doctors **
First Name, Second Name, Email, Phone, Role, and Gender.

** Doctors Record **
First Name, Second Name, Email, Phone, Role, and Gender.

** Add Outbreaks **
Outbreak, Comment, Location, and Measurers.

** Likely Outbreak **
Outbreak, Comment, Location, Measurers, and Edit.

** HIV **
Name, Age, Location, Mode of Contraction, Doctors Comment.

** HIV Reports **
Name, Age, Location, Mode of Contraction, Doctors Comment and Recorded.
